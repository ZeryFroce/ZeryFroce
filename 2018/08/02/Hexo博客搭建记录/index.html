Hexo博客搭建记录

发表于 2018-08-02 | 分类于 教程 |
字数统计: 727

博客，也就自己看的赏心悦目

现在人人都可以动手自己搭建一个博客的时代,Hexo+github page的方式极大的方便了过把博客瘾的快感，这个教程更多的是记录自己博客的搭建方式和设置，纯粹是自己的一个记录，以便下次提起裤子就可以跑！

准备工作

  • 注册Github，并建立一个库为：username.github.io [username]为你gtihub的名字
  • 下载安装git
  • 下载安装node.js
  • 有个域名（非必需）

    Hexo + Github page博客的搭建

    本地的搭建

  • 新建文件夹，例如Blog(下面都以此为例)，进入Blog右击运行Git Bash Here
  • 输入命令npm install -g hexo
  • 再运行 hexo init下载模板
  • 运行npm install
  • 最后启动本地服务器 hexo g和hexo s(本地打开localhost:4000即可看见博客的成功搭建了)

    与Github page关联

  • 打开Blog根目录下的_config.yml配置文件，找到deploy修改成如下;

  • 运行：npm install hexo-deployer-git –save

  • 运行：hexo g（本地生成静态文件）
  • 运行：hexo d（将本地静态文件推送至Github）
    此时访问：username.github.io 即为你的博客地址了。

    绑定域名

  • 在https://github.com/username/username.github.io仓库下新建一个文件CNAME,并填入域名，如

    examle.com

  • 到域名提供商处设置域名解析，添加以下纪录

  • 最后等待一段时间，即可享受自己的域名博客了

更新博客内容

  • 直接在git中hexo new "XX",即可打开Blog\source\_posts目录下建立的xx.md文件，进行码字了
  • 完成后执行hexo g和hexo s在本地预览
  • 本地检查无问题后即可执行hexo d同步到自己的博客上了
